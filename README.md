# 🏏 Pakistan Super League (PSL) 2025 Data Analysis

## 📌 Project Overview

This project presents an exploratory data analysis (EDA) of the Pakistan Super League (PSL) 2025 dataset using Python. The analysis focuses on understanding team performance, match outcomes, player statistics, and tournament trends through data visualization and statistical analysis.

The project demonstrates practical skills in data cleaning, exploratory data analysis, and visualization using real-world cricket data.

---

## 🎯 Objectives

- Clean and preprocess the dataset
- Explore match statistics
- Analyze team performance
- Study winning trends
- Identify top-performing players
- Create informative visualizations
- Generate insights from PSL 2025 data

---
